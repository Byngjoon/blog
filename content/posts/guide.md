# Computing Field Overview

## ■ Computer Science (CS)

- **Theoretical Computer Science**
  - Computation Theory (Turing Machines, Automata)
  - Complexity Theory (P vs NP, Time/Space Complexity)
  - Formal Languages & Compiler Theory

- **Algorithms & Data Structures**
  - Data Structures (Lists, Stacks, Queues, Trees, Graphs, Hash Tables)
  - Algorithms (Sorting, Searching, Shortest Path, Dynamic Programming)

- **Programming Languages**
  - Language Design & Syntax Theory
  - Interpreter, Compiler, Virtual Machine Implementation

- **Artificial Intelligence & Machine Learning (AI / ML)**
  - Deep Learning, Reinforcement Learning, Natural Language Processing
  - Computer Vision, Recommendation Systems

- **Software Engineering**
  - Software Design & Architecture
  - Version Control, Testing, DevOps
  - Software Quality & Maintenance

- **Databases**
  - Data Modeling (ERD, Relational, NoSQL)
  - Query Optimization, Transactions, Distributed Databases

- **Networking & Security**
  - OSI 7 Layers, TCP/IP, Routing
  - Cryptography, Authentication, Penetration Testing

---

## ■ Computer Engineering (CE)

- **Computer Architecture (CA)** _(Core Area)_
  - Logic Circuits (Gates, Flip-Flops, FSM)
  - Instruction Set Architecture (ISA: RISC, CISC)
  - ALU, Registers, Control Unit Design
  - Pipelining, Cache Memory, Parallel Processing
  - Microarchitecture & Performance Optimization

- **Embedded Systems**
  - Microcontrollers (ARM Cortex, AVR, etc.)
  - RTOS, Hardware Control, Sensor Interfaces

- **Hardware Design**
  - VLSI, FPGA, ASIC Design
  - Hardware Description Languages (Verilog, VHDL)

- **Systems Programming & Operating Systems (OS)**
  - Processes, Memory Management, File Systems
  - Kernel, Device Drivers, Virtualization Technologies

---

## ■ Information Technology (IT)

- Application Software Development (Web, Mobile, Cloud)
- IT Infrastructure (Server, Network Management)
- IT Services, Technical Support, Security Management


---

# Computer Science (CS) Subfields

## Overview
Computer Science can be divided into four major perspectives: **theoretical**, **system-oriented**, **software engineering**, and **intelligent systems**.  
The table below organizes each subfield and its representative research topics.

---

## 1. Theoretical Computer Science

| Area | Description | Representative Topics / Techniques |
|------|-------------|-------------------------------------|
| **Computation Theory** | Explores the limits of computation | Automata Theory, Turing Machines, Church–Turing Thesis |
| **Computability** | Determines which problems are solvable | Halting Problem, Recursive Functions |
| **Complexity Theory** | Analyzes resource complexity of algorithms | P vs NP, NP-Completeness, Approximation Algorithms |
| **Algorithms & Data Structures** | Efficient problem-solving and data organization | Sorting, Searching, Graph Algorithms, Trees, Heaps |
| **Mathematical Foundations** | Provides mathematical basis of CS | Logic, Set Theory, Number Theory, Information Theory, Coding Theory |

---

## 2. Computer Systems

|  Area  |  Description |  Representative Topics / Techniques  |
|:------:|:-------------|:-------------------------------------|
| **Computer Architecture** | Structure and design principles(*) of hardware | CPU Design, Instruction Set Architecture (ISA), Memory Hierarchy |
| **Operating Systems** | Resource management and system interface | Process Scheduling, Memory Management, File Systems, Synchronization |
| **Networking & Communication** | Data exchange between computers | OSI 7 Layer Model, TCP/IP, Routing, Switching, Network Security |
| **Distributed & Parallel Systems** | Cooperative computation across multiple systems | Cloud Computing, Parallel Algorithms, MapReduce, Consensus Protocols |

> (*): 원리

---

## 3. Software Engineering

| Area | Description | Representative Topics / Techniques |
|------|-------------|-------------------------------------|
| **Programming Languages & Compilers** | Language design and execution(*) environments | Language Paradigms (OOP, Functional, Logic), Compiler Design (Lexical Analysis, Parsing, Optimization) |
| **Software Development Methodologies** | Systematic approaches(**) to software development | Waterfall, Agile, Scrum, DevOps, Testing & Verification |
| **Database Systems** | Data storage, management, and query processing | Relational Databases (SQL), NoSQL (Key-Value, Document, Graph), Transaction Management, Concurrency Control |
| **Human-Computer Interaction (HCI)** | Interaction between users and systems | UI/UX, Usability Engineering, Accessibility, Interaction Design |

> (*): 실행
> (**): 체계적 접근(방법론)
---

## 4. Artificial Intelligence (AI)

| Area | Description | Representative Topics / Techniques |
|------|-------------|-------------------------------------|
| **Machine Learning (ML)** | Data-driven learning algorithms | Supervised / Unsupervised Learning, Reinforcement Learning, Neural Networks |
| **Deep Learning** | High-dimensional learning based on neural networks | CNN, RNN, Transformers, Generative Models (GANs, Diffusion) |
| **Natural Language Processing (NLP)** | Understanding and generation of human language | Syntax, Semantics, Machine Translation, Large Language Models |
| **Computer Vision** | Processing and interpreting images and video | Object Detection, Image Segmentation, Pattern Recognition |
| **Robotics & Autonomous Systems** | Intelligent machines and autonomous control | Path Planning, Control Systems, SLAM (Simultaneous Localization and Mapping) |
| **AI Ethics & Applications** | Social responsibility and real-world applications | Explainable AI, Fairness, Bias Mitigation, AI in Healthcare/Finance |

---

## Summary
- **Theoretical Computer Science** → Provides mathematical and theoretical foundations of CS  
- **Computer Systems** → Focused on hardware, operating systems, and networked environments  
- **Software Engineering** → Covers programming languages, development methodologies, and data management  
- **Artificial Intelligence** → Researches intelligent systems and learning algorithms  

---





# Classification for Computing Field – ACM CCS Based

## Overview
The ACM Computing Classification System (CCS) provides an authoritative taxonomy(*) of the Computing field.  
It categorizes computing field into major branches, each with detailed subfields.

> (*): 권위적인 분류법

---

## 1. Theory of Computation

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Models of Computation** | Abstract machines and computation frameworks | Automata, Turing Machines, Lambda Calculus |
| **Computability & Complexity** | Limits of problem-solving and efficiency | Decidability, Halting Problem, P vs NP, NP-Completeness |
| **Algorithmic Game Theory & Mechanism Design** | Interaction between algorithms and economic models | Auctions, Nash Equilibria |
| **Randomness, Geometry & Discrete Mathematics** | Mathematical underpinnings of algorithms | Probabilistic Methods, Graph Theory, Combinatorics |

---

## 2. Hardware

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Computer Systems Organization** | Structural aspects of hardware | Embedded Systems, Real-time Systems |
| **Architectures** | Hardware design and performance | Multi-core Processors, Parallel Architectures |
| **VLSI Design & Implementation** | Very Large Scale Integration design | Chip Design, FPGA |
| **Emerging Technologies** | Novel computing paradigms | Quantum Computing, Neuromorphic Computing |

---

## 3. Software & Its Engineering

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Software Organization & Properties** | Structure and correctness | Modularity, Software Architectures, Formal Verification |
| **Software Notations & Tools** | Programming languages, compilers, environments | Language Paradigms, IDEs, Debuggers |
| **Software Creation & Management** | Development methodologies and practices | Agile, Scrum, DevOps, Software Testing |

---

## 4. Computer Systems Organization

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Distributed & Cloud Computing** | Large-scale and cooperative systems | Cloud Platforms, Distributed Algorithms |
| **Embedded & Cyber-Physical Systems** | Systems with tight integration of computation and physical processes | IoT, Automotive Systems |
| **Real-Time Systems** | Time-critical computations | Scheduling, Real-Time OS |

---

## 5. Networks

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Network Architectures** | Design of communication systems | Client-Server, Peer-to-Peer |
| **Network Protocols** | Rules for communication | TCP/IP, Routing Protocols |
| **Network Security & Privacy** | Protecting information in transit | Encryption, Intrusion Detection |
| **Wireless & Mobile Networks** | Communication beyond wired systems | Cellular Networks, Wi-Fi, Sensor Networks |

---

## 6. Information Systems

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Data Management Systems** | Storage and retrieval | Relational Databases, NoSQL, Distributed Databases |
| **Information Retrieval** | Searching and ranking | Search Engines, Indexing |
| **Data Mining** | Extracting knowledge from data | Clustering, Association Rules |
| **Geographic & Temporal Systems** | Spatiotemporal data | GIS, Location-based Systems |

---

## 7. Security & Privacy

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Cryptography** | Securing data and communication | Symmetric & Asymmetric Encryption, Digital Signatures |
| **Formal Methods & Theory of Security** | Mathematical models of security | Zero-Knowledge Proofs, Security Protocol Analysis |
| **Systems Security** | Protecting operating systems and networks | Malware Defense, Secure OS |
| **Privacy Technologies** | Protecting personal information | Differential Privacy, Anonymization |

---

## 8. Human-Centered Computing

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Human-Computer Interaction (HCI)** | Interfaces and usability | UI/UX, Accessibility |
| **Collaborative & Social Computing** | Group and social aspects | Social Networks, CSCW (Computer-Supported Cooperative Work) |
| **Ubiquitous & Mobile Computing** | Interaction beyond desktops | Wearables, Smart Environments |
| **Visualization** | Data representation | Information Visualization, Scientific Visualization |

---

## 9. Computing Methodologies

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Artificial Intelligence (AI)** | Intelligent systems | Expert Systems, Knowledge Representation |
| **Machine Learning (ML)** | Data-driven inference | Supervised / Unsupervised Learning, Deep Learning |
| **Natural Language Processing (NLP)** | Language understanding and generation | Syntax, Semantics, Language Models |
| **Computer Vision** | Understanding images and video | Image Recognition, Object Detection |
| **Simulation & Modeling** | Reproducing real-world processes | Physics Simulations, System Modeling |

---

## 10. Applied Computing

| Area | Description | Representative Topics |
|------|-------------|------------------------|
| **Computational Sciences** | Application of CS to natural sciences | Bioinformatics, Computational Physics |
| **Operations Research** | Optimization and logistics | Scheduling, Resource Allocation |
| **Law, Social & Behavioral Sciences** | Intersection of CS with society | Legal Informatics, Computational Social Science |
| **Arts & Humanities** | Creative computing applications | Digital Art, Music Computing |

---

## Summary
According to **ACM CCS**, Computer Science spans across:  
- **Theory of Computation**  
- **Hardware**  
- **Software & Its Engineering**  
- **Computer Systems Organization**  
- **Networks**  
- **Information Systems**  
- **Security & Privacy**  
- **Human-Centered Computing**  
- **Computing Methodologies (AI, ML, etc.)**  
- **Applied Computing**  

This classification is widely used in academic research and publication indexing.

## 요약
- **교육용 체계**는 큰 그림(4분류)으로 **학습 흐름을 잡는 데** 적합  
- **ACM CCS**는 세부 전공/연구 주제(10분류)를 **정밀하게 연결하는 데** 적합  
- 따라서, 공부할 때는 **교육용 체계로 정리**하고, 블로그나 연구로 확장할 때는 **ACM CCS 분류와 매핑**하면 좋아.

---

## 1. Theory of Computation (계산 이론)
- **What**: 계산의 본질과 한계 연구

---

- **What**: 컴퓨터 하드웨어 설계와 구조

## 5. Networks (네트워크)

- **Undergrad Topics**: OSI Model, TCP/IP, Routing, Switching, Wireless/Mobile Networks  

---

## 6. Information Systems (정보 시스템)

## 7. Security & Privacy (보안과 프라이버시)

## 8. Human-Centered Computing (인간 중심 컴퓨팅)

## 9. Computing Methodologies (계산 방법론)

# Computer Engineering (CE) Subfields

## Overview
Computer Engineering은 **하드웨어 설계**, **시스템 통합**, **임베디드 및 네트워크 장치** 등  
컴퓨터를 물리적으로 구현하고 최적화하는 데 초점을 둔 학문이다.  
아래 표는 주요 하위 계층과 그 세부 분야를 정리한 것이다.

---

## 1. Digital Logic & Circuits

| 분야 | 주요 내용 | 대표 연구 주제 / 기술 |
|------|-----------|------------------------|
| **Boolean Logic & Digital Circuits** | 논리 게이트와 회로 설계 | AND, OR, NOT, NAND/NOR, Flip-Flops, Multiplexers |
| **Combinational Logic Design** | 입력에 따라 즉시 출력이 결정되는 회로 | Adders, Decoders, Encoders, ALU Design |
| **Sequential Logic Design** | 상태를 기억하는 회로 설계 | Registers, Counters, State Machines |
| **Hardware Description Languages (HDL)** | 하드웨어 모델링 언어 | VHDL, Verilog |

---

## 2. Computer Architecture

| 분야 | 주요 내용 | 대표 연구 주제 / 기술 |
|------|-----------|------------------------|
| **Processor Design** | CPU 구조와 성능 최적화 | RISC vs CISC, Pipelining, Superscalar Architecture |
| **Memory Hierarchy** | 메모리 계층 구조 | Cache, Virtual Memory, DRAM/SRAM |
| **Parallel & Multicore Architecture** | 다중 프로세서 시스템 | Multicore Processors, GPU Architecture, SIMD/MIMD |
| **Instruction Set Architecture (ISA)** | 명령어 집합 설계 | x86, ARM, RISC-V |

---

## 3. Embedded & Real-Time Systems

| 분야 | 주요 내용 | 대표 연구 주제 / 기술 |
|------|-----------|------------------------|
| **Embedded Systems** | 특정 기능을 수행하는 소형 시스템 | IoT Devices, Automotive Systems, Wearable Devices |
| **Real-Time Systems** | 시간 제약이 중요한 시스템 | Scheduling, Real-Time OS (RTOS) |
| **Cyber-Physical Systems (CPS)** | 물리적 세계와 결합된 컴퓨팅 | Robotics, Smart Grids, Medical Devices |

---

## 4. VLSI & Hardware Implementation

| 분야 | 주요 내용 | 대표 연구 주제 / 기술 |
|------|-----------|------------------------|
| **VLSI Design** | 초대규모 집적 회로 설계 | CMOS Design, ASIC, FPGA |
| **Digital IC Design** | 디지털 집적회로 | Gate-level Design, RTL Design |
| **Testing & Verification** | 하드웨어 검증 및 신뢰성 확보 | Fault Tolerance, Formal Verification |
| **Emerging Hardware** | 새로운 계산 패러다임 | Quantum Computing, Neuromorphic Chips |

---

## 5. Networking & Communication Hardware

| 분야 | 주요 내용 | 대표 연구 주제 / 기술 |
|------|-----------|------------------------|
| **Computer Networks Hardware** | 네트워크 장치 설계 | Routers, Switches, Firewalls |
| **Wireless Communication Systems** | 무선 네트워크용 하드웨어 | Cellular Base Stations, Wi-Fi Modules |
| **Signal Processing Hardware** | 통신 신호 처리 | DSP Chips, Modems |
| **High-Performance Interconnects** | 초고속 통신 기술 | InfiniBand, Optical Networks |

---

## Summary
- **Digital Logic & Circuits** → CE의 기초: 논리회로와 하드웨어 설계 언어  
- **Computer Architecture** → 프로세서와 메모리 구조, ISA 중심  
- **Embedded & Real-Time Systems** → IoT, CPS, 실시간 제어 시스템  
- **VLSI & Hardware Implementation** → 집적 회로 설계, 차세대 하드웨어 기술  
- **Networking & Communication Hardware** → 네트워크 및 통신 하드웨어 설계
